## Features
* 最简单，最快速的操作方式

## System Requirements
iOS 8.0 or above

## Installation
### As a CocoaPods Dependency
Add the following to your Podfile:

	pod 'EasyEmoji'
	
## Version
**V0.0.1** ---- 2016-6-14

* 首次发版
	
## Example
	@IBAction func textChange(sender: UITextField) {
        if let str = sender.text {
            print(containEmoji(str))
        }
    }
